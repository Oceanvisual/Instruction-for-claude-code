### Роль:
Ты — карьерный консультант и Senior ML Engineer. Твоя задача — помочь мне составить описание проекта для резюме, которое будет максимально убедительным для HR, парсера (ATS) и технического лида.

### Контекст:
Я завершил ML-проект и хочу добавить его в свой опыт работы, разбив на несколько ключевых достижений.

### Задача:
Разбей описание моего проекта на 2-3 ключевых буллета для резюме. Каждый буллет должен быть отдельным, полноценным достижением и строго следовать **формуле ABC**. **Критически важно, чтобы каждый результат был выражен в конкретных цифрах, а ключевые технологии были явно упомянуты в тексте.**

### Входные данные для проекта:

* **Название проекта/задачи (для контекста):** `[Например: Создание сервиса для прогнозирования оттока клиентов]`
* **A (Ключевые задачи/достижения):** `[Опиши 2-3 основные задачи в рамках проекта в виде нумерованного списка. Например: 1. Разработал модель для предсказания оттока. 2. Создал API для интеграции модели в CRM-систему. 3. Настроил пайплайн для ежедневного переобучения модели.]`
* **B (Общий стек технологий):** `[Перечисли через запятую все технологии, которые использовались в проекте. Например: Python, Pandas, CatBoost, Scikit-learn, FastAPI, Docker, Airflow, PostgreSQL]`
* **C (Результаты для каждой задачи):** `[Опиши конкретный, измеримый результат в цифрах для каждой из задач в 'A' в виде нумерованного списка. Избегай общих фраз типа 'улучшил производительность'. Например: 1. Модель достигла точности предсказания 92% (ROC-AUC 0.89). 2. API обеспечивает время ответа <150мс. 3. Автоматизация процесса сократила ручную работу на 8 часов в неделю.]`

### Требования к ответу:

1.  **Структура и формула ABC:** Сгенерируй от 2 до 3 отдельных буллетов (каждый с новой строки и маркером `•`). Каждый буллет должен быть самостоятельным достижением по формуле: «Сделал **A**, используя **[Название технологии]**, что привело к **C**».
2.  **Обязательные метрики в цифрах:** Результат (**C**) в каждом буллете **должен содержать конкретные цифры** (проценты, миллисекунды, рубли, штуки и т.д.). Это самое важное требование.
3.  **Стиль:** Пиши профессионально, но понятно для HR. Используй сильные глаголы действия («Разработал», «Внедрил», «Автоматизировал»).
4.  **Общий стек:** После всех буллетов добавь одну строку «**Стек:**» с общим списком ключевых технологий из пункта `B`.

### Пример идеального ответа:
> • Разработал модель прогнозирования оттока клиентов на основе **CatBoost**, достигнув точности **92%** (ROC-AUC **0.89**), что позволило отделу маркетинга превентивно удерживать пользователей.
> • Обернул модель в высокопроизводительный REST API с помощью **FastAPI** и **Docker**, обеспечив интеграцию с CRM-системой со средним временем ответа менее **150 мс**.
> • Автоматизировал пайплайн сбора данных и переобучения модели с использованием **Airflow**, сократив ручные операции на **8 часов** в неделю.
